## ✨ Características

- 📋 **Gestión completa de préstamos**: solicitudes, aprobación, desembolso, amortización, pagos, mora.
- 👥 **Clientes y sucursales** con jerarquía de usuarios y roles (RBAC).
- 📱 **WhatsApp integrado**: notificaciones de aprobación, recordatorios de pago, recibos automáticos.
- 🎨 **Marca blanca**: cada empresa imprime SU logo y datos en recibos, contratos, amortizaciones y estados de cuenta.
- 💾 **Respaldos automáticos** cifrados a USB y/o Google Drive Desktop con organización por fecha.
- 🛡️ **Auto-recuperación** ante corrupción de base de datos.
- 🔐 **Licencias anti-piratería** vinculadas al hardware del cliente (Ed25519).
- 🔄 **Auto-actualizaciones** sin reinstalar manualmente.
- 📊 **Reportes en PDF** profesionales (recibos, amortización, contratos, estados de cuenta).
- 📤 **Exportación a Excel** desde todos los módulos.
- 🇩🇴 **Pensado para República Dominicana**: RD$, RNC, sistema dominicano de teléfonos.
